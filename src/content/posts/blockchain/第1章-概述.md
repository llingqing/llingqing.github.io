# 第1章 概述 · 考点速记

> 区块链是伴随比特币诞生的、以密码学为基础的去中心化分布式账本技术；本章讲清它的由来、定义、三要素、四大特点、智能合约与世界计算机、三种分类，以及应用、挑战与发展环境。

## 🎯 核心考点速览
- **区块链概念源于中本聪（Satoshi Nakamoto）2008 年比特币白皮书**《比特币：一种点对点电子货币系统》，但白皮书并未给出"区块链"的明确定义。
- **区块链 = 区块（Block）+ 链（Chain）**：区块是数据基本结构单元，链是由区块产生的哈希链表。
- **三大基本要素：交易（Transaction）、区块（Block）、链（Chain）**。
- **四大特点：去中心化、透明性、不可篡改性、多方共识**。
- **不可篡改的根本原因**：哈希链表 + 消息摘要（改一页 → 摘要不匹配 → 需连带改后续所有区块并获全网承认），配合"挖矿难度"提高篡改门槛。
- **三种分类：公有链、联盟链、私有链**（开放程度依次递减）；另一分类：许可链 / 无须许可链。
- **智能合约概念由 Nick Szabo 提出，远早于区块链**；以太坊（Vitalik Buterin）将其落地并提出"世界计算机（World Computer）"。
- 区块链底层技术：**P2P 网络、分布式系统、密码学、共识机制**（不含机器学习）。

## 1.1 什么是区块链技术

### 定义
- **狭义**（工信部 2016《中国区块链技术和应用发展白皮书》）：按时间顺序将数据区块以顺序相连方式组合成的链式数据结构，并以密码学方式保证**不可篡改、不可伪造**的**分布式账本**技术。
- **广义**：利用块链式数据结构验证与存储数据、分布式节点共识算法生成更新数据、密码学保证传输访问安全、自动化脚本（智能合约）编程操作数据的**全新分布式基础架构与计算范式**。
- 本质：分布式存储、点对点传输、共识机制、加密算法等的新型应用模式；是**价值互联网**的基石，在互不信任环境中实现去信任中介的可信交易。

### 三大基本要素
- **交易（Transaction）**：一次操作，导致账本状态的一次改变。
- **区块（Block）**：记录一段时间内的交易和状态结果，是对当前账本状态的一次共识。
- **链（Chain）**：由区块按发生顺序串联而成，是整个区块链状态变化的日志记录。
- 通俗比喻：区块链是记录所有历史交易的账本，每个区块是账本的一页；每页记录上一页的摘要信息。

### 1.1.1 比特币与区块链的诞生（加密货币先驱脉络）
- **1983 David Chaum**：论文《Blind Signatures for Untraceable Payments》，提出基于盲签名的 **e-Cash**（匿名、难追踪）。
- **1996 e-gold**（Douglas Jackson、Barry Downey）：以金银担保、瞬时转账；2009 年前后因洗钱/勒索/黑客退市。
- **1997 Adam Back — Hashcash**：用于反垃圾邮件和 DoS，提出**工作量证明（PoW）**机制，被后来数字货币采用。
- **1998 戴伟（Wei Dai）— B-money**：首个完全不依赖中心机构的匿名数字货币方案，引入 PoW 和初步 P2P 网络；但未解决"双花"和账本安全，未实现。
- 其他尝试：WebMoney、Liberty Reserve、Perfect Money 等均未成功（中心化节点被突破即崩溃）。
- **关键时间**：2008.10.31 中本聪在 metzdowd.com 密码学邮件列表发布白皮书；**2009.1 比特币网络正式上线，中本聪挖出创世区块，获首批 50 个比特币**；2010 年有人用 **10000 个比特币买 25 美元披萨**（比特币首次交易、首次拥有交换价值）。

### 1.1.2 比特币与区块链（关系）
- 比特币是区块链**第一个/最成功的典型应用**；区块链是**比特币的底层技术**，二者不等同。
- "区块链"一词诞生**晚于**比特币，但核心概念、结构、原理已在中本聪设想中提及。
- 中本聪解决"无权威第三方却让所有人承认交易"的两大措施：**区块链结构 + 挖矿**。
- **挖矿**：为提高篡改门槛，规定区块哈希值须达到规定"难度"，求解符合要求的随机数的过程；篡改需对后续区块逐个重算哈希，越久的记录越难改，难度随时间**指数级上升**。
- 区块链不仅是比特币：本质是去中心化数字账本，可记录任何有价值、不希望被篡改的信息（支付汇款、衍生品、征信、数字资产、审计、供应链金融、物联网、产品溯源、信息存证等）。

### 1.1.3 区块链的特点（四大特点）
1. **去中心化**：无权威中心机构，任意个人/组织可参与挖矿、交易、验证；少量节点离线不影响整体；避免高成本、易欺诈、缺乏透明、滥用权限。
2. **透明性**：账本分发给全网所有参与者，交易与历史对持有者公开透明（区别于用户匿名性）。
3. **不可篡改性**：交易记录在链上，中心机构无法自行修改，比特币账本很难被篡改。
4. **多方共识**：多方参与维护须约定校验/写入/冲突解决规则，即**共识算法**；比特币用 **PoW**，联盟链共识更灵活贴近业务。

### 1.1.4 智能合约与世界计算机
- **智能合约概念由 Nick Szabo（计算机科学家、密码学专家）提出，远早于区块链**：将现实合约/契约引入数字领域约束参与方行为、惩罚违约。
- **以太坊 Vitalik Buterin**：用区块链实现智能合约约束条件，无需外部力量；智能合约转变为"运行在区块链上的计算机程序"。
- 以太坊 = 随系统运行永不停歇的智能合约分布式平台 = 一台去中心化的计算机，Vitalik 称之为**世界计算机（World Computer）**。
- 意义：智能合约 + 以太坊是区块链发展的**里程碑事件**，使区块链真正成为基础底层架构。

### 1.1.5 区块链的分类
- **按访问权限开放程度**：公有链（Public）、私有链（Private）、联盟链（Consortium）。
- **按节点准入许可**：许可链（Permissioned）、无须许可链（Permissionless）。
- 开放程度：**公有链 > 联盟链 > 私有链**（依次递减）；公链最公平但慢、效率低；联盟链/私链效率高但削弱去中心化。

## 1.2 区块链技术的现状

### 1.2.1 区块链的应用
- **比特币**：哈希算法为主的密码技术；PoW 控制产出速度，总量趋近 **2100 万个**；每区块 **1MB**、最多约 2000 多笔交易、约 **10min 出块**；速度 **3~7 笔/秒**（VISA 峰值 24000 笔/秒），易堵塞、手续费虚高、算力浪费。
- **以太坊**：Vitalik Buterin（发布白皮书时仅 19 岁）；2014 落地并募资 1.5 亿美元，2015 系统完成；市值第二的数字货币系统 + 智能合约平台；吞吐量约 **10~20 笔/秒**；共识为**权益证明（PoS）**；经历多次分叉。
- **联盟链**：使区块链与加密数字货币剥离。**2015.12 Linux 基金会启动 Hyperledger 项目**；**Hyperledger Fabric** 是较完备的联盟链平台，具保密性/可伸缩/灵活/可拓展，可控制用户权限（通道 channel）、可选共识（SOLO、Kafka、SBFT 等）。
- 应用领域（图1.1）：数字货币、供应链、政务公开、司法存证、金融、物联网等。
- 《2018 中国区块链产业白皮书》：截至 2018.3 底，我国以区块链为主营业务的公司达 **456 家**；招商银行、民生银行等自 2016 年起探索金融应用。

### 1.2.2 区块链的挑战
- **Gartner 技术成熟度曲线（Hype Cycle）** 五阶段：促动期 → 顶峰期（过高期望）→ 低谷期（泡沫化）→ 光明期（稳步爬升）→ 高原期（实质生产/稳定应用）。**2018 年区块链已越过顶峰期、进入低谷期**。
- **技术层面**：
  1. **效率问题**（比特币约 7 笔/秒，难满足支付场景）；
  2. **共识机制问题**（难兼顾安全与效率，部分有逻辑漏洞，PoW 浪费算力）；
  3. **数据冗余问题**（每节点存完整链，数据不断增长挑战存储）；
  4. **安全问题**（智能合约语言不成熟、漏洞多；著名 **The DAO 事件损失约 6000 万美元**）。
- **法律层面**：
  1. **匿名化问题**（难监管追踪，可绕开外汇管制，威胁经济基础）；
  2. **欺诈问题**（认知低+炒作，滋生欺诈，如**庞氏骗局**）；
  3. **冲击铸币权**（加密货币试图取代法定货币，威胁国家经济民生政权稳定）。

### 1.2.3 区块链技术的发展环境
- **国外起步早、整体水平高**。
- **美国**：国土安全部、NIST、NSF 等提供资金/技术支持；至少 8 个州在制定区块链立法；纳斯达克用区块链记录私人证券交易（伦敦、香港证交所响应）；桑迪亚国家实验室开发比特币去匿名化工具；斯坦福、普林斯顿、加州伯克利、MIT 等开课（MIT 试点用区块链签发毕业证书）。
- **欧洲**：22 个欧洲国家签署区块链合作伙伴关系声明；欧洲央行 2017 年表示不限制区块链创新技术。
- **亚洲**：柬埔寨、印尼等建支付/小额信贷/供应链项目；**日本承认比特币合法地位**并允许部分交易；韩国政策激进（Bangco 与 SNK 合作 BangcoChain，高丽大学与 Tobesoft 合作）。
- **中国**：清华、北大、浙大、中山大学等开课设研究中心；《"十三五"国家信息化规划》将区块链列入战略性前沿技术；杭州建首个区块链产业园、设首个百亿规模创新基金；广州黄埔区出台"区块链 10 条"；**2019.10.24 中共中央政治局就区块链进行第十八次集体学习**。

## 1.3 本书的内容
- 当前区块链技术体系：以 **P2P 网络、分布式系统、密码学、共识机制**为主，多种改良技术为辅。
- 各章：第2章比特币基础（哈希/非对称加密/交易/UTXO 模型/脚本/公私钥地址/区块结构/共识）；第3章以太坊（账户/交易/数据结构与存储）；第4章网络层（P2P、网络层安全）；第5章共识层（分布式理论、拜占庭容错、PoW 及新共识）；第6章智能合约（Solidity、DApp、Solidity OJ）；第7章区块链应用（供应链金融、票据流通、司法存证、资产交易、物流溯源）。

## ⚖️ 重点对比 / 易混区分

### 公有链 / 联盟链 / 私有链（表1.1）
| 对比项 | 公有区块链 | 联盟区块链 | 私有区块链 |
| --- | --- | --- | --- |
| 控制者 | 所有参与者 | 特定的联盟成员 | 链所有者 |
| 去中心化程度 | 强去中心化 | 弱去中心化、多中心化 | 无 |
| 交易吞吐量 | 小于 100 笔/秒 | 最高约 100000 笔/秒 | 由配置决定 |
| 可修改性 | 几乎不可篡改 | 较难被篡改 | 可被篡改 |
| 共识机制 | PoW、PoS、DPoS 等 | PBFT、Raft 等 | 无特定算法 |
| 代表项目 | 比特币、以太坊、EOSIO 等 | Hyperledger Fabric、企业以太坊 | 无特定项目 |
